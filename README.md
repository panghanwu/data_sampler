# Video Sampler

This script helps you sample frames from your target video.

## Requirements

- Python 3
- numpy
- scikit-learn
- opencv-python

## Main Features

- All frames extracting
- Uniform sampling
- Uniformly random sampling
- K-means sampling 
- Sampling by detector (better for bounding-box form)
