# Data Sampler

This script helps you sample:

- frames from your target video
- Images from a bunch of images

## Requirements

- Python 3
- numpy
- scikit-learn
- opencv-python

## Main Features

### `VideoSampler`

- All frames extracting
- Uniform sampling
- Uniformly random sampling
- K-means sampling 
- Sampling by detector (better for bounding-box form)
